# DGX Spark / Thor Testing Guide

## Prerequisites

- Docker installed (`docker --version`)
- Test audio files downloaded (`python3 tests/download_test_audio.py`)
- Image built (`docker build -t birdnet-species:0.1.1 .`)

## Running Tests

### 1. Quick Smoke Test

```bash
docker run --rm \
  -v $(pwd)/tests/audio:/data \
  birdnet-species:0.1.1 \
  --input /data/search_sample.mp3 --dry-run
```

Expected: Blue Jay detected at 99.7% confidence.

### 2. Multi-Species Soundscape

```bash
docker run --rm \
  -v $(pwd)/tests/audio:/data \
  birdnet-species:0.1.1 \
  --input /data/soundscape.wav --dry-run
```

Expected: Black-capped Chickadee (81%), Dark-eyed Junco (74%),
House Finch (64%), American Goldfinch (48%).

### 3. With Geo-Filtering (Chicago, Week 22)

```bash
docker run --rm \
  -v $(pwd)/tests/audio:/data \
  birdnet-species:0.1.1 \
  --input /data/soundscape.wav --dry-run \
  --lat 41.88 --lon -87.62 --week 22
```

### 5. CSV Output

```bash

  -v $(pwd)/tests/audio:/data \
  birdnet-species:0.1.1 \
  --input /data/soundscape.wav --dry-run \
  --output /data/results.csv

# Check results
cat tests/audio/results.csv
```

### 6. All Test Audio (Batch)

```bash
for f in tests/audio/*.wav tests/audio/*.mp3; do
  echo "=== $(basename $f) ==="
  docker run --rm \
    -v $(pwd)/tests/audio:/data \
    birdnet-species:0.1.1 \
    --input "/data/$(basename $f)" --dry-run \
    --min-confidence 0.40
  echo ""
done
```

## Performance Benchmarks

Record inference time for each file to establish baselines:

```bash
for f in tests/audio/*.wav tests/audio/*.mp3; do
  echo -n "$(basename $f): "
  docker run --rm \
    -v $(pwd)/tests/audio:/data \
    birdnet-species:0.1.1 \
    --input "/data/$(basename $f)" --dry-run 2>&1 \
    | grep "Classified" | sed 's/.*in //' 
done
```

### Expected Performance (DGX Spark / Thor ARM64)

BirdNET V2.4 uses TFLite on ARM64 (CPU-only). Benchmarks:
- 3-second audio chunk: ~50-100ms per chunk
- 15-second recording: ~300ms total inference
- 2-minute soundscape: ~3-4s total inference

These are significantly faster than Raspberry Pi 4, which already
runs real-time. The DGX Spark / Thor ARM64 cores are much more
powerful.

## Deployment on Sage/Waggle (Thor H00F)

```bash
# Build and tag for pluginctl
docker build -t birdnet-species:0.1.1 .

# Deploy with pluginctl (when ready)
sudo pluginctl deploy \
  --name birdnet-species \
  birdnet-species:0.1.1 \
  -- --duration 15 --lat 41.7 --lon -87.99 --week 25 \
     --min-confidence 0.25 --interval 60
```
